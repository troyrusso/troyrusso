## 🔬 Featured Research

### [Weighted Greedy Sampling (WiGS)](https://github.com/thatswhatsimonsaid/WeightedGreedySampling)
*Co-developed active learning algorithm currently submitted to UAI 2026.*
* **The Problem:** State-of-the-art active learning heuristics for regression (like iGS) rely on a static, multiplicative balance between feature space and predictive output space, which fails in heterogeneous domains.
* **The Solution:** We formulated the selection strategy as a Reinforcement Learning (RL) problem using a Soft Actor-Critic (SAC) agent to dynamically modulate the weight between exploration and exploitation.
* **The Impact:** Achieved up to a **40% reduction in RMSE** on complex synthetic datasets and a nearly 50% reduction on real-world benchmarks compared to the state-of-the-art iGS baseline. 
* **Tech:** Python, SLURM, HPC Clusters, scikit-learn, Reinforcement Learning.
